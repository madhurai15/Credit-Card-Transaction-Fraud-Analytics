## Credit Card Transaction & Fraud Analytics
An end-to-end data analytics project that analyzes credit card transaction data using **Python, SQL, and Power BI** to uncover spending patterns, detect fraudulent transactions, and generate actionable business insights.
# Project Overview
This project demonstrates the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL querying, and interactive dashboard development. The dashboard enables users to monitor transaction performance, customer behavior, and fraud trends.

# Dataset
The project uses a publicly available **Credit Card Transactions Dataset** from Kaggle.
 **Note:** The cleaned dataset is not included in this repository due to its large size (approximately 400 MB). All preprocessing steps are available in the Jupyter Notebook, allowing the dataset to be reproduced from the original data.
**Dataset:** *(https://www.kaggle.com/datasets/priyamchoksi/credit-card-transactions-dataset)*
